A high-\/performance digital audio playback simulation that leverages custom-\/built data structures, multithreaded background processing, and real-\/time audio manipulation using the SFML library.

This project emulates a classic i\+Pod-\/style interface, focusing on efficient playlist management, seamless playback, and optimized memory usage.

-\/-\/-\/

\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+Overview\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}

The system is designed to handle large music libraries efficiently by combining:\+

-\/ Fast Navigation — Constant time O(1) track transitions ~\newline


-\/ Memory Efficiency — Custom template-\/based data structures ~\newline


-\/ Continuous Playback — Background thread automation ~\newline


-\/-\/-\/

\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+Objectives\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}

-\/ Provide constant-\/time navigation between tracks ~\newline


-\/ Ensure memory safety through custom data structures ~\newline


-\/ Maintain uninterrupted playback using concurrent execution ~\newline


-\/-\/-\/

\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+Core Features\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}

\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+Custom Doubly Linked List Architecture\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}

-\/ Bidirectional traversal using a template-\/based \doxylink{class_doubly_linked_list}{Doubly\+Linked\+List$<$t$>$} ~\newline


-\/ O(1) transitions between next and previous tracks ~\newline


-\/ Deep copy constructor for safe playlist duplication ~\newline


-\/ Dynamic insertion and deletion without full reallocation ~\newline


\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+Multithreaded Playback Engine\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}

-\/ Background monitoring using std::\+thread ~\newline


-\/ Thread-\/safe control via std::\+atomic$<$bool$>$ ~\newline


-\/ Automatic transition to the next track upon completion ~\newline


-\/ Clean shutdown without race conditions ~\newline


\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+Persistent Storage System\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}

-\/ CSV-\/based file storage for playlists ~\newline


-\/ Efficient serialization of metadata (title, artist, duration, file path) ~\newline


-\/ Persistent state across program executions ~\newline


\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+Real-\/\+Time Audio Control (SFML)\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}

-\/ Fine-\/grained seeking using set\+Playing\+Offset ~\newline


-\/ Efficient audio streaming directly from disk ~\newline


-\/ Reduced memory consumption compared to full buffering ~\newline


\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+Playback Modes\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}

-\/ Random shuffle using indexed traversal (O(n)) ~\newline


-\/ Circular playlist navigation (tail to head and vice versa) ~\newline


-\/ Continuous playback without interruption ~\newline
